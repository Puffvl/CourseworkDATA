База данных автопроката 
9 таблиц 
1. cars        содержит основные данные по авто 
2. car_brand   как часть таблицы cars содержит дополнительные данные такие как марка авто и страну производитель.
3. car_types   тоже содержит дополнительные сведения тип кузова атомобиля.
разбил на три  части для того что бы не повторялись одни и те же данные в общей таблице , а только их ID.
4. car_status  содержит список статусов автомобиля.
5. car_service содержит список автомастерских
6. rent        содержит информацию о аренде авто
7. maintance   содержит информацию о ремонтах и техническом обслуживании авто
8. customer    информация о клиентах
